## Quick start 🏄

Prerequisites: [Node (v19 LTS)](https://nodejs.org/en/download/) and [Git](https://git-scm.com/downloads)

> Clone the repository:

```
git clone https://github.com/salviega/cross-carbon-frontend.git
```

> Install dependencies:

```
yarn
```

> Run the project:

```
yarn dev
```
